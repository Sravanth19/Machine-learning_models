# ü§ñ Machine Learning Models & Algorithms

This repository serves as a comprehensive collection of Machine Learning and Deep Learning implementations. It covers the entire data science pipeline, from data preprocessing to building complex models like Neural Networks and XGBoost. Each folder contains Python scripts and datasets designed to help users understand the core logic behind these algorithms.

## üìÇ Repository Structure

The project is organized into logical sections based on different machine learning domains:

* **Part 1 - Data Preprocessing**
    * Handling missing data, categorical data, splitting datasets, and feature scaling.
* **Part 2 - Regression**
    * Predicting continuous values using Linear Regression, SVR, Decision Trees, and Random Forests.
* **Part 3 - Classification**
    * Categorizing data using Logistic Regression, K-NN, SVM, Kernel SVM, Naive Bayes, and Random Forests.
* **Part 4 - Clustering**
    * Unsupervised learning using K-Means and Hierarchical Clustering.
* **Part 5 - Association Rule Learning**
    * Finding relationships in data using Apriori and Eclat models.
* **Part 6 - Reinforcement Learning**
    * Upper Confidence Bound (UCB) and Thompson Sampling.
* **Part 8 - Deep Learning**
    * Artificial Neural Networks (ANN) and Convolutional Neural Networks (CNN).
* **Part 9 - Dimensionality Reduction**
    * Principal Component Analysis (PCA), Linear Discriminant Analysis (LDA), and Kernel PCA.
* **Part 10 - Model Selection & Boosting**
    * k-Fold Cross Validation, Grid Search, and XGBoost.
